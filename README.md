# Sadaqah Jariyah Impact Calculator

An interactive calculator showing both the worldly and eternal impact of Sadaqah Jariyah donations — with verified quotes from the Qur'an and Hadith showing actual numbers.

## Features

- **Interactive Slider**: Adjust donation amount to see real-time impact
- **Two Modes**: Water wells and masjids, each with verified impact data
- **Worldly Impact**: People served, years of service, total litres/acts of worship
- **Eternal Reward**: 700× minimum multiplier (Qur'an 2:261) with 3 authenticated proof cards per mode
- **Verified Sources**: All Islamic quotes sourced from Sahih al-Bukhari, Sahih Muslim, Sunan al-Nasā'ī, etc.
- **MATW Brand Theme**: Magenta (#f60362) and blue (#00a3da) gradient design
